🏗️ SYSTEM ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════════

Request Flow:
    User Code
        ↓
    POST /analyze (Flask)
        ↓
    kb.services.fix.analyze_and_fix()
        ↓
    rag.orchestrator.analyze()
        ├─ Defect prediction (local)
        ├─ Knowledge retrieval (FAISS)
        └─ Fix generation (smart router)
            ↓
        rag.llm.generate_fix()
            ├─ TRY: Gemini API (3-10s)
            ├─ FALLBACK: Local LLM (if available)
            └─ FALLBACK: Heuristics (instant)
            ↓
        Result Masking
            └─ Shows as "Qwen2.5-Coder-1.5B-Instruct"
            ↓
        Templates Render
            ↓
        User Receives Code Fix
